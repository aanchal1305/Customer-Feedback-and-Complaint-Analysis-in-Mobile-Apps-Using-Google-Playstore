# Customer-Feedback-and-Complaint-Analysis-in-Mobile-Apps-Using-Google-Playstore
A data-driven analytics platform that extracts actionable insights from user reviews of mobile applications using the Google Play Store dataset. 
This project integrates Exploratory Data Analysis (EDA), Natural Language Processing (NLP), and Machine Learning to identify complaint patterns, analyze sentiment, and predict app success.

Built using Python, Streamlit, Scikit-learn, and NLP techniques, the system enables stakeholders to:

1) Analyze user sentiment distribution across apps and categories
2) Detect complaint patterns such as crashes, lags, and feature issues
3) Perform interactive EDA on app metadata (ratings, installs, pricing)
4) Generate install recommendations using predictive modeling
5) Visualize insights through an interactive dashboard
